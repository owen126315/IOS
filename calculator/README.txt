Additional functions:

1: Error checking

   If the answer of the equation can not be calculated or have syntax error with in the equation, the answer label will show "error". If "=" button is pressed at that instance, an alert dialog will popped up and tells what error it is, such as an operator appeared at the end of the equation, brackets are not in pairs, the answer is complex number.

2: Memory storage

   There are two buttons "MS" and "MR". For "MS", it stores the text on the equation text field when it is pressed. And for "MR", it recall the saved text and insert in to the equation text field.
